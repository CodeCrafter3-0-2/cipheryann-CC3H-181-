OPEN INNOVATION
Problem Statement- In today’s digital world, users are logged into multiple websites simultaneously (social media, banking, emails, etc.). This creates serious privacy and security risks, especially when:

Users forget to log out from shared or public devices
Sessions remain active across multiple tabs and websites
Unauthorized access can occur due to saved cookies and sessions
There is no centralized way to log out from all platforms instantly

Existing browsers do not provide a single-click global logout system, forcing users to manually log out from each website, which is inefficient and unsafe.
Solution- To solve this problem, we propose a browser extension called “SecureSurf” that provides instant global logout functionality.
